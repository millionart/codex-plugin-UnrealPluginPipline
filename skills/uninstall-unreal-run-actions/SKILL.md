# Uninstall Unreal Run Actions

Use this skill when the user says `uninstall` for Unreal Plugin Pipeline or explicitly wants the built-in Run dropdown actions removed.

## Workflow

1. Find the target `.uplugin` under the current workspace.
2. Locate this plugin bundle's `scripts/unreal-plugin-pipeline.mjs`.
3. Run:
   `powershell -NoProfile -ExecutionPolicy Bypass -File "<bundle-root>/scripts/invoke-unreal-plugin-pipeline.ps1" -PipelineCommand uninstall -ProjectRoot "<project-root>"`
4. Report the project-local environment file that was changed or removed.

## Safety

- Remove only the `UNREAL PLUGIN PIPELINE ACTIONS` marker block.
- Preserve unrelated existing Run actions.
- Delete an environment file only when it was generated by this plugin and no other actions remain.
- Do not modify parent/workspace `.codex/environments/environment.toml` files outside the target plugin project root.
